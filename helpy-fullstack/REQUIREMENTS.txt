# Helpy Helpdesk - Complete Requirements List
# This file lists all required software, packages, and extensions

## ============================================
## SYSTEM REQUIREMENTS
## ============================================

### Operating System
- Windows 10/11, macOS, or Linux
- Minimum 4GB RAM (8GB recommended)
- 2GB free disk space

### Required Software
1. Node.js v18.0.0 or higher
   - Download: https://nodejs.org/
   - Verify: `node --version`

2. npm v9.0.0 or higher (comes with Node.js)
   - Verify: `npm --version`

3. PostgreSQL v12.0 or higher
   - Download: https://www.postgresql.org/download/
   - Default password used: root123
   - Verify: `psql --version`

4. Git (optional but recommended)
   - Download: https://git-scm.com/downloads
   - Verify: `git --version`

## ============================================
## NOTES
## ============================================

- All packages are managed via npm (Node Package Manager)
- Backend runs on port 4000 by default
- Frontend runs on port 5173 by default
- Database tables are auto-created by Sequelize on first run
- File uploads are stored in backend/uploads/ directory
- JWT tokens are used for authentication
- CORS is enabled for frontend-backend communication
